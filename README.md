# Grapecity

**index.html**
This is the main web page of the project.
It contains:
-Header Section: Contains the main title of the site.
-Navigation Icons: Two linked icons (GitHub & LinkedIn) encouraging visitors to explore profiles.
-Resume Link: A button or link that prompts users to download or view the resume PDF.
All of this is styled using CSS for layout, typography, and visual appeal.

**style.css**
This file contains all the styling rules for the web page, including Fonts,Colors and themes,Icon positioning,Layout adjustments (e.g., centering content),Hover effects for links and buttons.
